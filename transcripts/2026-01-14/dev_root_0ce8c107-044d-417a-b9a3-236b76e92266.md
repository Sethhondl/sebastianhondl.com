To fix this, try the following:

**Re-enable mouse mode:**
1. Press your tmux prefix key (usually `Ctrl+b`)
2. Then press `:`
3. Type: `set -g mouse on` and press Enter

**Or from outside tmux**, run:
```
tmux set -g mouse on
```

If that doesn't work, you might be stuck in **copy mode**. Press `q` or `Escape` to exit copy mode and return to normal scrolling.

```

```
